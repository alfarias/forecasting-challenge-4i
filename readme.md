# 4Intelligence Forecasting Challenge

## Introduction
The hallenge is divided in two cases: one about the TFP rate of USA, Canada and Mexico, and another about brazilian products export.<br>
The development was made with Python 3.7 using modules like Pandas for data processing, hvplot for charts and forecastings with Prophet. All the modules used are on the `requirements.txt` file, which can be used to reproduce the code. Jupyter Notebooks are used to development.<br>

## Folders
* `data`: Contains the main data used for the forecasts.
* `docs`: Files with instructions for the tasks and support documents.
* `models`: Saved models from Prophet.
* `notebooks`: Notebooks used for the Cases 1 and 2, there are additional files for these cases in html format to read the content on any web browser.
* `references`: Additional files with data and documents to help on forecasts.
* `reports`: Figures from the charts and csv files with the forecasts.
